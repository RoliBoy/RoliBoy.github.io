archer

automated arch linux install script

Features

Support for multiple DE/WM

choose from multiple desktop environment and window manager options

Friendly interface

menu-based TUI installer

Hardware detection

for CPU, GPU, boot mode, battery, wireless, bluetooth and SSD

Automatic system configuration

nvidia optimus backends, tlp and systemd configurations

Mirror ranking

download speed and ping based ranking

Progress bar

installation progress feedback

Usage

installation steps:

  • download the official archiso
  • make a bootable media
  • boot
  • run curl -Ls roliboy.ml/archer | bash
  • follow installation dialog
